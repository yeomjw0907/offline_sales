# 화면 구성

## 1. 공통

- 인증 방식: 카카오 로그인
- 권한: 파트너 / 일반 관리자 / 최고관리자
- 공통 화면:
  - 로그인
  - 접근 권한 없음
  - 개인정보 처리방침
  - 이용약관

## 2. 파트너 화면

### 2.1 신청 화면

- 기본 정보 입력
- 개인정보 수집·이용 동의
- 광고성 정보 수신 동의
- 제출 후 승인대기 안내

### 2.2 파트너 홈

- 내 추천인 코드
- QR
- 이번 달 실적
- 누적 실적
- 예상 수수료
- 정산 예정 금액
- 지급완료 금액
- 최근 실적 내역

### 2.3 영업자료

- Ready Talk 바로가기
- 서비스 소개 자료
- 리플릿 PDF
- 영업 스크립트
- 코드 사용 안내
- 문의 링크

### 2.4 정산 정보

- 계좌정보 등록/수정
- 정산 상태 확인
- 지급 내역 조회

## 3. 관리자 화면

### 3.1 관리자 대시보드

- 전체 파트너 수
- 승인대기 수
- 이번 달 실적 건수
- 누적 실적 건수
- 정산 예정 금액
- 최근 신청/실적 목록

### 3.2 파트너 관리

- 파트너 목록
- 상태 필터
- 승인/비활성 처리
- 추천인 코드 확인
- 파트너 상세 보기

### 3.3 실적 관리

- 실적 직접 등록
- 등록 실적 목록
- 추천인 코드 검색
- 중복 의심 경고
- 실적 수정/취소

### 3.4 정산 관리

- 정산 예정 대상 목록
- 파트너별 정산 금액 확인
- 3.3% 공제 계산 결과 확인
- 지급완료 처리

### 3.5 자료 관리

- 파트너용 자료 링크 등록/수정
- Ready Talk 링크 관리
- 리플릿/스크립트 관리

### 3.6 관리자 관리

- 일반 관리자 관리
- 최고관리자 전용 권한 작업
- 관리자 액션 로그 확인
