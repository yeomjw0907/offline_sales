# 서비스 기획

## 1. 서비스 개요

- 서비스명: Ready Talk 영업 파트너 모집 및 운영 시스템
- 목적: 파트너 모집, 승인, 추천인 코드 발급, 실적 집계, 정산 관리를 한곳에서 운영한다.
- 운영 주체: 98점7도(원케이션)
- 연동 대상: Ready Talk

## 2. 해결하려는 문제

- 파트너를 모집해도 실적 귀속과 정산 관리가 수기로 흩어지면 운영이 빠르게 복잡해진다.
- 파트너별 고유 코드와 실적 현황이 정리되지 않으면 동기부여가 떨어지고 분쟁 가능성이 커진다.
- 초기에는 고객사 API 연동 없이도 빠르게 시작할 수 있는 운영 시스템이 필요하다.

## 3. 핵심 사용자

### 파트너

- 카카오 로그인으로 가입
- 신청 정보 입력 후 승인 대기
- 승인 후 개인 추천인 코드/QR 수령
- 내 실적, 예상 수수료, 지급완료 금액 확인
- 영업자료와 Ready Talk 바로가기 링크 활용

### 관리자

- 파트너 승인/비활성 처리
- 추천인 코드 관리
- 실적 직접 입력
- 정산 대상 확인 및 지급 처리

### 최고관리자

- 일반 관리자 권한 관리
- 귀속 수정
- 코드 재발급
- 정산 최종 처리 및 민감 정보 접근

## 4. MVP 범위

- 파트너 카카오 로그인
- 파트너 신청 폼 제출
- 관리자 승인 후 추천인 코드 발급
- 파트너 전용 대시보드
- 관리자 실적 직접 입력
- 추천인 코드 기반 실적 반영
- 정산 예정/지급완료 관리
- 영업자료 링크 노출

## 5. 제외 범위

- Ready Talk API 자동 연동
- 카카오톡 안에서 실적 조회
- 자동 알림톡 발송
- 실적 자동 대조
- 고급 리포트/분석 대시보드

## 6. 핵심 가치 제안

### 파트너 입장

- 복잡한 영업 시스템 없이 추천인 코드만으로 활동할 수 있다.
- 내 실적과 정산 현황을 투명하게 확인할 수 있다.
- 영업자료와 링크를 한곳에서 받는다.

### 운영자 입장

- 파트너 모집부터 승인, 실적, 정산까지 같은 시스템에서 관리할 수 있다.
- 초기에는 수기 입력으로 빠르게 시작하고, 추후 자동화로 확장할 수 있다.

## 7. 핵심 흐름

1. 파트너가 카카오 로그인 후 신청 정보를 제출한다.
2. 관리자가 파트너를 승인한다.
3. 시스템이 고유 추천인 코드와 QR을 발급한다.
4. 파트너가 오프라인/온라인에서 코드를 배포한다.
5. 자영업자가 Ready Talk 파일럿 시작 시 추천인 코드를 입력한다.
6. 관리자가 파일럿 시작 건을 직접 등록한다.
7. 파트너 대시보드와 관리자 대시보드에 실적이 반영된다.
8. 매월 10일 기준 정산이 진행된다.
